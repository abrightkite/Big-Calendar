# Wellcome My React big calendar Challenge Repo!
React big Calendar 공식 문서에 있는 모든 APIs를 써보고 정리해보자!

## React Big Calendar
**react-big-calendar** 는 React와 최신 브라우저용으로 제작된 이벤트 일정관리 컴포넌트이다.
**react-big-calendar** 는 `**flexbox`** 를 사용한다.
